Mijn ervaringen

Op deze website toon ik al mijn ervaringen, leg ik uit wat ik heb moetten doen en hoe deze mij geholpen hebben om te groeien.

Stage gemeentehuis Rijkevorsel

In mijn laatste jaar in het middelbaar heb ik een stage van 3 weken gelopen bij het gemeentehuis van Rijkevorsel.
Ik heb hier niet al te speciale opdrachten gekregen maar deze waren elke keer op andere plekken van de geemnte. Omdat het
verschillende sectoren waren heb ik geleerd om sociaal te zijn op een profesionele manier en heb ik ook geleerd hoe de IT
veel delen van een gemeente laat werken en hoeveel werk er in gaat.

Studentenjob Wienerberger

Ik heb al 3 zomers een studentenjob gehad bij het steen fabriek Wienerbereger. Deze waren altijd in Augustus en waren een maand lang.
Het eerste jaar heb ik doorheen de fabriek klusjes gedaan en 2 weken als machine operator gewerkt.
De 2 andere zomers heb ik op de IT afdeling gewerkt. Hier heb ik machines moetten na tekenen en deze bruikbaar moetten maken
door er vb code aan toe te voegen. Ik heb ook drivers moetten controleren doorheen de fabriek en back ups gemaakt van
besturing programmas voor machines gemaakt. Dit was zeer leerrijk doordat ik in programma's heb moetten werken dat ik nog nooit had gezien.
Mijn verantwoordelijke heeft ook veel uitleg gegeven over servers, hoe hij de fabriek in werking blijft en hoe machines blijven runnen.
Ik heb hier ook kunnen mee lopen met een codeerder van een machine dat moest geinstaleerd worden. Dit was ook zeer leuk zodat
ik in real time een codeerder bezig zag en hoe het op de werkvloer er aan toe gaat.

Action breebos

Ik heb een deel in de zomer bij de action gewerkt. Dit zou ik normaal niet toevoegen omdat dit gewoon achter de kassa zitten was.
Maar er wonen zeer veel buitenlander rondom breebos. Dit heeft me hard geholpen om klanten te helpen dat niet dezelfde taal spreken.
Dit is volgens mij zeer handig om te kunnen omdat je altijd klanten kunt hebben dat niet dezelfde taal spreken of zeer weinig.
